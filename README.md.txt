# SesAura 🔊

## 📌 Project Overview
SesAura is a smart acoustic alert system designed to assist hearing-impaired individuals.  
It detects important environmental sounds and provides real-time alerts through mobile notifications and LED indicators.

---

## 🚀 Features
- Real-time sound detection
- Detects:
  - Baby crying 👶
  - Fire alarm 🔥
  - Door knock 🚪
  - Glass breaking 🪟
  - Dog bark 🐶
  - Human scream 😱
- Mobile notifications with vibration
- LED alert system
- Speech-to-text conversion

---

## 🛠️ Technologies Used
- Python
- TensorFlow Lite
- Firebase Realtime Database
- Raspberry Pi
- YAMNet (Sound classification model)

---

## ⚙️ How It Works
1. Captures audio using microphone
2. Processes audio using YAMNet model
3. Classifies sound using trained model
4. Sends alerts via Firebase
5. Displays alerts on mobile app and LED
